# KnifeTruck

수원 기준 이동형 칼갈이 트럭 제보와 지역 알림 흐름을 검증하는 MVP 프로젝트입니다.

## Public Scope

이 저장소에는 공개 가능한 최소 정보만 포함합니다.

- 서비스 한 줄 설명
- 공개용 프로젝트 소개
- 향후 제품 방향의 아주 높은 수준 요약

내부 기획 문서, 데이터 수집 자료, 운영용 스키마, 비공개 실험 자료는 저장소에서 제외합니다.

## Product Summary

핵심 아이디어:
- 사용자가 동네에서 칼갈이 트럭을 발견하면 빠르게 제보한다.
- 최근 제보를 지역 기준으로 확인한다.
- 관심 지역을 저장하고 신규 제보 알림으로 다시 돌아오게 만든다.

문제:
- 칼갈이 트럭은 이동형이라 언제 어디에 오는지 알기 어렵다.

해결 방향:
- 제보 기반 위치 공유
- 지역 중심 탐색
- 관심 지역 알림
